## Hints

If you don't know how to move forward, here some tips that might help you.

### Filtering a collection

To filter a collection you can use the filter method. Here how to use it https://developer.mozilla.org/fr/docs/Web/JavaScript/Reference/Objets_globaux/Array/filter 

### React Hooks

The React Hooks are really the swiss knife of React, they can be useful for example to keep a component in a certain state. You can read more about that here https://reactjs.org/docs/hooks-intro.html 

### Forms

How to interact with Forms in React ? Don't worry we got your back: https://reactjs.org/docs/forms.html
